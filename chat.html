The Challenges: to create a chat room application similar to Slack.

My work: to create a chat room application website As the front-end web programmer, my job was to use the Firebase web service to do authentication and server-side data storage/synchronization and use the react-router to create different pages or chanels.

Description: KCALS is a chat room application I built. The name is meaningless except for it is the reversed version of SLACK. On this website, users must sign up with their emails and get a email verification. They then can create different channels. Within each channel, users can post, edit, like, and delete their own posts and only like other people’s posts. The time of the post is recorded.

What I learned:
● Practicing to use React framework to build an interactive web page.
● Using Firebase to include user authentication and synchronized data storage in your app.
● Using client-side routing to create single-page applications
● Utilizing third-party JavaScript libraries in building a web application
