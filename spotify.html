The Challenges:to create an interactive music website that enables users to interactively explore information from a public Web API

My work: : to build a music website that hat utilizes web API and React, a full-features web framework. As the front-end web programmer, my job was to use React to create an interactive music website by using a public Web API and React to produce data-driven information services. In order to have the website to interactively produce information, I also created new DOM elements in the page based on data I receive from the server.

Description: Mini Spotify is an interactive website that utilizes Spotify API and React to create a platform to give users a chance to explore new artists and possibly like them by listening those artists’ 30-second song previews. The song previews are taken from Spotify API. Once the user click on the “preview” button, a music player will pop out with a randomly selected song playing. The user can adjust the volume and the time of the song and skip a song.
