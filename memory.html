The Challenges: to create a web-based version of Memory & Concentration.

My work: to build an interactive, engaging and web-based memory game by utilizing new libraries As the front-end web programmer, my job was to integrate HTML, CSS, JavaScript, and existing libraries, such as jQuery, Bootstrap, Lodash, and Moment.

Description: Memory & Concentration is a web-based game that starts with a grid of cards (8 pairs) turned face down and the user flips two cards in a round. If the two cards flipped are matched, they stay flipped. If the two cards flipped are not matched, the two cards flip back and the user is required to remember where each card is in order to find the matched ones. The playing time is recorded and the user will be praised after they finish.

What I learned:
● Combining HTML, CSS, and JavaScript to create interactive and engaging website
● Dynamically manipulating DOM contents and attributes
● Event driven logic of programs implementation
● Discovering and finding the most suitable libraries and utilizing their functions
