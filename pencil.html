The Responsive Chanllenge

The Challenges: to make a responsive website to fit in all screen sizes.

My work: to build an accessible, responsive, and modern-styled mock commercial website As the front-end programmer, my job was to write HTML and CSS style rules to utilize Flexbox and make the website responsive to the screen size it is rendered on.

Description: This is a mock commercial pencil website that could fit in any electronic screens, no matter computers, ipads, or mobiles. Almost every commercial product has a website to advertise for their products. It is very important for a website to fit in any screen sizes. The purpose of this website is to demonstrate my ability to utilize Flexbox through CSS style rules to make the website responsive and suitable for all screen sizes, and to utilize different CSS style rules to adjust the font sizes, paddings, and links, and backgrounds for different screen sizes.

What I learned: Writing semantically rich HTML website
Utilizing Flexbox to make the website responsive
Utilizing CSS style rules to adjust font style, font sizes, backgrounds, and links.
