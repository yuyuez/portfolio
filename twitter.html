The Challenges: to make an interactive website that provides users feedback when users send requests through interactions like searching in search bar.

My work: to make an interactive, accessible, and responsive Twitter posts analysis website As the front-end web programmer, my job was to write JavaScript to process and manipulate data

(Search item: Seahawks)

Description: Tweeter Sentiment is a website that does simple sentiment analysis of Twitter data and determine the attitude of tweets made by a particular Twitter user. It takes real data from Twitter users’ timeline and analyze the mood of that Twitter user. For example, we search the word “seahawks” and find 4.52% of the words are positive, such as happy and love, and it shows top three popular related hashtags that seahawks provides.

Value: According to the research study “Twitter mood predicts the stock market”, emotion can influence decision making and behaviors. Thus, emotions shown in tweets can also reflect what people would do in stock market. This website is meant to help analyze moods of Twitter users, which could indirectly predict the stock market. *Reference: Bollen, Johan, Huina Mao, and Xiaojun Zeng. "Twitter Mood Predicts The Stock Market". Journal of Computational Science 2.1 (2011): 1-8. Web.

What I learned:
● Using JavaScript and programming skills to iterate over data and analyze data
● Dynamically generating DOM content from data and adding it to a web page
● Managing asynchronous programming by using callback functions and promises
